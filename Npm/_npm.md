## Npm

#### NPM（node package manager）
* 通常称为node包管理器，管理node包
* 包括: 安装、卸载、更新、查看、搜索、发布等。
* npm的背后，是基于couchdb的一个数据库，详细记录了每个包的信息，包括作者、版本、依赖、授权信息等。
* 它的一个很重要的作用就是：将开发者从繁琐的包管理工作（版本、依赖等）中解放出来，更加专注于功能的开发。
* 链接：https://npmjs.org/
