## GitBook

#### 开源工具链
GitBook 是一个使用 Git 和 Markdown 来构建书籍的工具。它可以将你的书输出很多格式：PDF，ePub，mobi，或者输出为静态网页。

#### GitBook.com
GitBook.com 是一个使用工具链来创建和托管书籍的在线平台 [www.gitbook.com](https://www.gitbook.com)。
